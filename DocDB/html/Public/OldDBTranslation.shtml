Translations to New Database

What is this page?

You have probably come to this page looking for a reference in the old BTeV document database numbering scheme. These references look like BTeV-pub-Year/# while the numbering scheme for the new document database looks like BTeV-doc-####-v#. If you are searching for a document whose number you don't know or have a reference to the new style of numbering, go to the new Document Database to find the document.

If you have an old style number, find it in the lists here and click on it, you will be taken to the corresponding document in the new database.
