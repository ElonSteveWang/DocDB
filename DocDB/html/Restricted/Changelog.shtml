Document Database Changes
This page lists the changes to the document database code over time.

7 October 2002

  • ShowDocument uses the "asof" parameter. Probably won't do the same for RetrieveArchive
      4 October 2002
      • Requesting files from a document by the date of modification is now possible. The "asof" parameter will pick the current version as of a date. I.e. asof=2002-06-01. Still have to do the same for ShowDocument and RetrieveArchive
          3 October 2002
          • Fixing some security issues. Users can now be flagged as able to view but not create documents (for reviewers, etc.). Also, there were some problems with people being able to update documents they couldn't access if they knew what parameters to pass in. These should be fixed.
              22 July 2002
              • RetrieveFile now supports retrieving files by extension.
              15 July 2002
              • Fixed (in stable) a problem where OBSOLETE versions of Revisions where being picked up.
              • Renamed the function FetchDocRevision to FetchRevisionByDocumentAndVersion
              9 July 2002
              • Devel version allows changing of meta-info for old documents. Warnings are given before and after the user does this.
              2 April 2002
              • Devel version updated so that all functions return arrays rather than references to arrays. This change should have no visible impact.
              • New code is under evaluation for next stable version
              1 April 2002
              • Preferences activated for stable version
              27 March 2002
              • Reorganized menus
              • Less verbage on submitting a document successfully
              • Descriptions are preset on update as suggested by Lynn
              • Preferences are mostly complete in devel version
