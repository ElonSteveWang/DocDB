This document lays our a rough road map of where DocDB should be headed in the next months. As always, this is subject to modification due to BTeV's needs and the needs of those contracting for improvements.

  • 6.0 release
    • Add certificate based authentication and with that the concept of individuals that belong to groups.
    • Add CSS (Cascading Style Sheet) framework and a new CSS based layout for ShowDocument.
  • 6.x releases
    • Individual document notification.
    • Single session meetings (simplified creation).
    • Document cross-referencing.
    • Move over to single ListBy script, deprecate, then delete others.
    • CSS for more scripts.
  • 7.0 release
    • Truly separate meetings and topics
