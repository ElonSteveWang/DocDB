Linux RPMS

Install the following rpms (needed by various parts of DocDB)

  • MySQL rpms if you will have a local database
    • mysql-devel
    • mysql-server
    • mysql
    • mysql-client
  • MySQL rpms if the database is located on another machine
    • mysql
    • mysql-client
  • libdbi
  • libdbi-dbd-mysql
  • libdbi-devel
  • perl-DBI
  • perl-DBD-MySQL
  • perl-DBD-Pg
  • perl-XML-Grove
  • perl-MailTools
    (downloaded from http://www.rpmfind.net/linux/RPM/contrib/noarch/)
  • perl-Digest-SHA1
    (downloaded from http://www.rpmfind.net/linux/RPM/contrib/noarch/)
  • macutils
  • zip
